# captures

This directory stores optional packet captures (`.pcap`) and traffic logs collected during the lab.

## Suggested Captures

| File | Description |
|------|-------------|
| `vlan10-ping.pcap` | ICMP traffic between PC1 and PC3 (VLAN 10) |
| `vlan20-ping.pcap` | ICMP traffic between PC2 and PC4 (VLAN 20) |
| `trunk-tagged.pcap` | 802.1Q tagged frames on the trunk link |

Add your capture files here and update this table accordingly.
